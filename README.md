# README

## Prerequisites

` gem install bundler`

## Insall application dependencies

`bundle install`

`rake db:create db:migrate `

## Run the server

` rails s -b 0.0.0.0`

Open in the browser

localhost:3000

## Run tests

### All tests

`rspec`

### Individual tests

`rspec spec/controllers/stories_controller_spec.rb `
